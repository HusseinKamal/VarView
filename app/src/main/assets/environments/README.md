# Environment HDR Files

Place your HDR environment map here for SceneView's Image-Based Lighting (IBL).

## Required File
- `studio_small.hdr` — A studio-style HDR environment for realistic lighting.

You can download free HDR environment maps from:
- https://polyhaven.com/hdris
- https://ambientcg.com/

Recommended: A neutral indoor studio HDRI for clothing visualization.
